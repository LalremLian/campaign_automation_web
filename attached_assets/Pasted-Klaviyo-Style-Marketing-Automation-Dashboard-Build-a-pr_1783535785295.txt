Klaviyo-Style Marketing Automation Dashboard

Build a premium, modern SaaS web application inspired by Klaviyo. This is a clickable UI prototype with mock functionality only. The goal is to impress clients with a clean, professional interface that demonstrates the product vision. No real backend is required—use mock data, local state, and realistic interactions.

* shadcn/ui

* Framer Motion
